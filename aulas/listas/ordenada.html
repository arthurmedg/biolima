Passos para Criar um HTML

  1. Criar um novo arquivo .html
  2. Escrever "!" para gerar a estrutura básica
  3. Mudar a línguagem para a sua preferência
  4. Mudar o título
  5. Linkar o HTML com o CSS no head
  6. Adicione o conteúdo básico dentro do body
  7. Use a tag h1/2/3 para fazer os títulos e subtítulos
  8. Adicione um p (parágrado) para escrever um texto
  9. Salve o arquivo
  10. Faça o commit do arquivo

Checklist Entregando uma Atividade

  1. Salve o arquivo que será entregue com um nome intuitivo e adequado
  2. Abra o link da Atividade
  3. Confira se está na atividade correta
  4. Clique em "+ Adicionar ou criar"
  5. Selecione o método que será utilizado
  6. Procure o Arquivo
  7. Selecione o Arqivo
  8. Confira se selecionou o arquivo certo
  9. Envie o arquivo
  10. Espere pela nota ou correção do exercício
